- `Khoshaba_Odeesho_Resume.pdf` — downloadable CV

## ⚠️ Known issue to fix

The `canonical`, Open Graph (`og:url`, `og:image`), and Twitter Card meta tags in `index.html` currently point to a placeholder domain (`khoshaba-odeesho.example.com`) rather than the real hosting URL. This should be replaced with the actual live URL once the site is deployed — until then, link previews shared on LinkedIn/Twitter/etc. will show broken or incorrect metadata, and search engines will index the wrong canonical URL.

## Deployment

Static HTML — deployable as-is via GitHub Pages, Vercel, or Netlify.
